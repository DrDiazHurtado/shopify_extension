**Shopify Insider** is a high-performance Chrome Extension designed for Data Scientists, E-commerce Researchers, and Market Analysts. It enables one-click metadata extraction from any Shopify store, revealing pricing trends, inventory recency, and best-selling products.

---

## üöÄ Key Capabilities

- **Automated Metadata Extraction**: Instantly fetch product titles, variants, pricing, and vendor details.
- **Smart Competitor Auditing**: Reveal the "Hero Product" dominance and catalog concentration.
- **CSV Engine**: Export clean, structured data for immediate integration with Pandas, Excel, or SQL databases.
- **Strategic Insights**: Detect psychological triggers like Urgency (FOMO) and Bundling (AOV optimization).
